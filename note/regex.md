# Regular Expression
## 元字符(metacharactors)
| .          | 单个任意字符   |
| []         | 字符组         |
| [^]        | 排除型字符组   |
| \char      | 转义           |
| ?          | 匹配1次或0次   |
| *          | 匹配0次或多次  |
| +          | 匹配至少1次    |
| {min, max} | 区间次数       |
| ^          | 一行的开头     |
| $          | 一行的结束     |
| \<         | 单词的开头     |
| \>         | 单词的结束     |
| |          | 匹配任意分隔的表达式 |
| ()         | 限制匹配的范围 |
| \1, \2     | 反引用括号内容 |
